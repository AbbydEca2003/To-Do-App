# 📝 Simple To-Do List App

Yet another minimalist web-based To-Do List app to help you keep track of your daily tasks and stay organized.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using LocalStorage
- Responsive design

## 🛠️ Technologies Used

- Java
- Android Studio
- LocalStorage (SQLite)
